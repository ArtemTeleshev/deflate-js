deflate-js: DEFLATE in JavaScript

deflate-js is a pure JavaScript implementation of the DEFLATE algorithm. DEFLATE is a popular compression algorithm used both as a stand-alone compression and in the popular GZIP file format.

Overview

With HTML5, complex web application are being written, and there is still no good way to compress data in the browser. The W3 specification gives no way to call native code from a client-side application and most browser APIs are hidden as well. One solution is to make a browser extension to do the heavy lifting, but this obviously requires users to install the extension just to use functionality. This is, of course, not preferred, and this project tries to solve that problem.

Usage

deflate-js is built to work with pakmanager, a package builder for the browser. This means that it follows node-style requires: 

    
      var DEFLATE = require('deflate-js');

      // compress a string
      var compressed = DEFLATE.deflate('Hello World');

      // decompress a compressed string
      var decompressed = DEFLATE.inflate(compressed);
