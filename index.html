deflate-js: DEFLATE in pure JavaScript

deflate-js is a pure JavaScript implementation of the DEFLATE algorithm.

DEFLATE is a popular compression algorithm used both as a stand-alone compression and in the popular GZIP file format.

Use Cases

  • Text editors (compress to reduce bandwidth)
  • Generating compressed tarballs
  • Pushing server-side tasks involving compressed data to the client
  • Show off

Usage

deflate-js is built to work with pakmanager, a package builder for the browser. This means that it follows node-style requires: 

    
  var DEFLATE = require('deflate-js');

  // compress a string
  var compressed = DEFLATE.deflate('Hello World');

  // decompress a compressed string
  var decompressed = DEFLATE.inflate(compressed);
